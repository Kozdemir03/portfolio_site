Perfect Circle

How perfect can you draw a circle freehand?

HTML5 JavaScript Canvas API Chart.js

Overview

Perfect Circle is a browser game that challenges you to draw the most accurate freehand circle possible. When you lift your pen or mouse, the app instantly calculates a score from 0 to 100 % using centroid-distance standard deviation analysis — the lower the deviation of each point from the ideal radius, the higher your score.

Key Features

  • Canvas-based freehand drawing with real-time score calculation
  • Centroid + standard deviation algorithm for accuracy measurement
  • Role-based auth: admin / regular user with sessionStorage sessions
  • Persistent localStorage leaderboard — top scores survive page reload
  • Admin dashboard with Chart.js score distribution, bar charts, and stats
  • Responsive layout — works on desktop and tablet

How the Scoring Works

After you finish drawing, the app collects all recorded points, computes the centroid, then calculates the average distance (r̄) from the centroid to each point. The standard deviation of those distances is used to quantify irregularity. A perfectly round path would have σ = 0, giving a score of 100 %. The final score formula maps σ to the 0–100 range with a smooth sigmoid-like curve so that small improvements still feel rewarding.

Challenges

  • Smoothing raw mouse/touch coordinates to avoid sampling noise without sacrificing responsiveness.
  • Designing a score formula that feels fair at both extremes — a nearly perfect circle should score 99+, while a rough oval shouldn't score above 50.
  • Building a lightweight auth system without a backend — solved with hashed passwords in sessionStorage and a hard-coded admin role flag.
  • Keeping Chart.js lightweight — only the bar and scatter chart modules were imported.
