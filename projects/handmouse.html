Mouse-HandTracking

Hands-free mouse control via webcam hand tracking

Python MediaPipe OpenCV PyAutoGUI

Overview

Mouse-HandTracking lets you control your computer mouse entirely with hand gestures captured by a standard webcam — no physical mouse needed. Cursor movement, left/right clicks, and scroll are all mapped to hand landmarks detected in real time.

The project also ships as a system-tray application with Windows startup Registry integration, so it launches automatically on boot. A one-click installer and a live camera-preview UI make setup painless.

Key Features

  • Real-time hand-gesture recognition at 30+ fps via MediaPipe Hands
  • Cursor movement mapped to index-finger landmark position
  • Pinch gesture for left-click, two-finger pinch for right-click
  • Vertical palm movement for scroll
  • System tray icon — enable/disable without opening the window
  • Windows Registry startup integration
  • Live camera preview window with landmark overlay
  • One-click installer (PyInstaller bundle)

Tech Stack

Built entirely in Python, leveraging Google's MediaPipe for hand-landmark detection and PyAutoGUI for cross-platform mouse control. OpenCV handles webcam capture and the preview window rendering.

Python 3.x MediaPipe Hands OpenCV PyAutoGUI pystray (system tray) winreg (startup) PyInstaller

Challenges

  • Keeping cursor movement smooth despite per-frame landmark jitter — solved with an exponential moving-average filter on x/y coordinates.
  • Distinguishing intentional click gestures from natural hand tremor — thresholded on finger-distance delta over multiple frames.
  • Packaging a Python app with webcam access into a single distributable exe — handled via PyInstaller with hidden-imports for MediaPipe's compiled modules.
  • Reliable Registry startup entry that survives Windows updates and UAC prompts.
