KampYeri

Full-stack campsite & hotel reservation platform

C# ASP.NET SQL Bootstrap

Overview

KampYeri is a production-ready reservation platform built at F1Proje (Oct 2024 – Apr 2025) that allows end-users to search, filter, and book campsites, hotels, and caravan areas across Turkey.

Businesses can register, create dynamic listings with photos and pricing, manage bookings, and track revenue through a dedicated admin dashboard — all from within the same application.

Key Features

  • User registration, login, and role-based access (guest / business / admin)
  • Dynamic business listings with availability calendars
  • Date-range booking with conflict detection
  • Business management portal — add/edit venues, photos, pricing
  • Admin dashboard with booking and revenue statistics
  • Responsive Bootstrap frontend
  • Standalone admin panel for a pre-existing web application (separate deliverable)

Architecture & Tech

The backend is built on ASP.NET Web Forms / MVC with C#. Data access uses LINQ-to-SQL, keeping queries type-safe and refactorable. The database schema was designed from scratch, normalised to 3NF, and includes foreign keys for bookings, listings, users, and media assets.

C# ASP.NET LINQ-to-SQL SQL Server Bootstrap 5 jQuery

My Role

  • Designed the full SQL schema — entities, relationships, indexes, and stored procedures.
  • Built the LINQ data-access layer, eliminating raw SQL strings throughout the codebase.
  • Implemented user authentication, password hashing, and session management.
  • Developed the business-registration and listing-management module end-to-end.
  • Built a separate, standalone admin panel for an existing company web application using C#, LINQ, and SQL.
