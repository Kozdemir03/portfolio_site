Mobile App Analysis

Data analysis on 1,500+ apps from Google Play & App Store

Python Pandas Seaborn Web Scraping

Overview

An end-to-end data analysis project covering the full pipeline from data collection to visual insight. The dataset — 1,500+ apps scraped from both Google Play and the Apple App Store — was processed, cleaned, analysed, and visualised to surface meaningful patterns about mobile app markets.

Pipeline Stages

  • Scraping: custom Python scrapers collected app metadata (ratings, installs, size, category, price) from both stores.
  • EDA: shape, dtypes, value counts, correlation matrix.
  • Imputation: platform-specific strategies for missing values (median for numerics, mode for categoricals).
  • Outlier removal: IQR method for skewed columns, Z-Score for normally distributed ones.
  • Analysis: 10 GroupBy and conditional Pandas queries.
  • Visualisation: 10 chart types with Seaborn and Matplotlib.

Visualisations Produced

The project generated 10 distinct chart types to communicate different aspects of the data:

Histplot Heatmap (correlation) Pairplot Violinplot Boxplot Barplot (GroupBy) Countplot Scatterplot KDE plot Stripplot

Tech Stack

Python 3.x Pandas Seaborn Matplotlib BeautifulSoup / Requests (scraping) Jupyter Notebook NumPy
